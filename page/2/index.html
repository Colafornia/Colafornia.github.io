  • Thrift 速记

    Mar 27, 2018

    基本概念

    Apache Thrift 是一款 RPC (跨语言的服务)框架，传输数据采用二进制格式，相对 XML 和 JSON 体积更小，对于高并发、大数据量和多语言的环境更有优势。

    RPC 是什么

    Remote Procedure Call 即远程过程调用。

    RPC 是一个 计算机通信协议。该协议允许运行于一台计算机的程序调用另一台计算机的子程序，而程序员无需额外地为这个交互作用编程。 —— 中文维基
